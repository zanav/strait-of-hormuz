# Strait of Hormuz

A tiny browser game inspired by the Strait of Hormuz news cycle — because sometimes the world’s most stressful “open/closed” toggle deserves a game.

Built with Claude Code as a fully self-contained HTML app.

## Play

Open `index.html` in your browser.

## Why?

The news from home has been heavy lately, and I wanted a small, playful distraction from constant doomscrolling.

This is not meant to be serious — just a tiny AI-built experiment that brought me a bit of joy during a stressful news cycle.
